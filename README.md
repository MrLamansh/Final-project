# Исследование эффективности вакцинации в борьбе с COVID-19


Исследование измеряет, насколько массовая вакцинация повлияла на исходы пандемии COVID-19:
снизилась ли летальность, кто из стран оказался лидером в борьбе с вирусом, какие внешние
факторы (возраст населения, богатство, система здравоохранения) объясняют различия между
странами и как пандемия сказалась на экономике.

## Ключевые выводы

- Вакцинация измеримо снизила летальность. При сравнении стран «до/после» начала прививок
  медианная летальность (CFR) упала примерно втрое - с 1.8% до 0.7%, снижение наблюдается у
  126 из 134 стран (критерий Уилкоксона, p = 2·10⁻²⁰).
- Наивные межстрановые корреляции обманчивы. В сырых данных высокий охват вакцинации
  коррелирует с более высокой смертностью (+0.37) из-за конфаундинга возрастом и полнотой
  учёта. После контроля этих факторов связь становится отрицательной (−0.20), как и ожидалось.
- Лидеры по сочетанию признаков «высокий охват + низкая летальность + надёжные данные» -
  Сингапур, Южная Корея, Новая Зеландия, Австралия.
- Сезонность есть, но не доминирует: зимняя смертность выше летней на 40%, однако динамику
  определяли прежде всего штаммовые волны, а не календарь.
- Вакцинация связана с импульсом восстановления 2021 года, но не с итоговым
  двухлетним результатом (эффект отскока).

## Источники данных

- Our World in Data (OWID) - случаи, смерти, вакцинация, демография и социально-экономические
  показатели: <https://github.com/owid/covid-19-data>
- Всемирный банк, World Development Indicators - годовой рост ВВП (`NY.GDP.MKTP.KD.ZG`),
  подключение через API: <https://api.worldbank.org>


Очищенные наборы данных (`covid_clean_daily.csv`, `covid_clean_summary.csv`) и основной датасет
OWID (`owid-covid-data.csv`, 94 МБ) генерируются автоматически при запуске файла и
поэтому не хранятся в репозитории.

## Этапы работы

1. Сбор данных - загрузка OWID, первичный осмотр.
2. Очистка и предобработка - отсев агрегатов, фильтрация состава (152 страны), обработка
   пропусков, 7 новых расчётных признаков.
3. Разведочный анализ (EDA) - распределения, корреляции, выявление конфаундеров.
4. Проверка гипотез - статистические тесты H1–H6 (Уилкоксон, Спирмен, регрессия).
5. Временные ряды - STL-разложение (тренд/сезонность), прогноз Хольта–Уинтерса.
6. Экономический блок - обогащение через API Всемирного банка, проверка H4.
7. Выводы - определение лидеров, синтез результатов.

## Как запустить

```bash
pip install -r requirements.txt
jupyter notebook covid_project.ipynb
```

Затем выполнить ячейки по порядку («Run All»). При первом запуске файл скачает датасет OWID
(94 МБ) и обратится к API Всемирного банка.
